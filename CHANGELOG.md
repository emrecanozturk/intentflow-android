# Changelog

## 0.1.0

Initial Android architecture proposal.

Added:

- IntentFlow core runtime for Kotlin and Android-style app layers
- typed state, intent, event, effect, output, and route model
- pure reducer contract
- coroutine-backed store
- effect cancellation IDs and policies
- projection contract for UI-facing state
- trace helper for behavior tests
- IntentFlow AI manifest model
- manifest validator
- feature generator CLI
- AI context generator for Codex, Claude, Gemini, Copilot, Cursor, and generic tools
- Compose adapter example
- ViewModel adapter example
- MVVM migration example
- pattern research matrix
- wiki source pages
- Cursor, Claude, Gemini, Copilot, and Codex instruction surfaces
- CI, CodeQL, docs workflow, Dependabot, issue templates, discussion templates, security policy, contribution guide, and release checklist
